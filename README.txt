# GabbrovMod
> Experimenting around with Minecraft modding — a hands-on extension of skills built in game engine development.
---
## About
GabbrovMod is a personal Minecraft mod project focused on learning and expanding skills in areas like **particle systems**, **shaders**, and game modding architecture. It draws directly from experience gained while building a [3D Voxel Engine](https://github.com/Gabbrov1/Game-Repo) in C#, applying those concepts in a new environment.
---
## Background
### Previous Work
This project is a direct continuation of two earlier efforts:
1. **3D Voxel Engine** — A C# project that laid the groundwork for understanding 3D rendering, voxel-based worlds, and game systems. ([View Repository](https://github.com/Gabbrov1/Game-Repo))
2. **Python TkInter Mod Tool** — A GUI tool built to accelerate the generation of modded blocks, since each new block or item in Minecraft traditionally requires editing approximately **4 JSON files** and **3 Java files** (sometimes more depending on setup). This tool automated much of that boilerplate. Unfortunately, it was lost along with the original GitHub account.
GabbrovMod picks up where those projects left off, rebuilding momentum in the modding space after an extended break.
